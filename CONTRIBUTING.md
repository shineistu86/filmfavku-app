# Panduan Kontribusi - FilmFavku

## Cara Berkontribusi

1. Clone repository
2. Buat branch baru: `git checkout -b fitur/nama-fitur`
3. Lakukan perubahan
4. Commit dengan format: `tipe(scope): deskripsi`
5. Push ke branch Anda
6. Buat Pull Request

## Format Commit

- `feat`: Penambahan fitur baru
- `fix`: Perbaikan bug
- `docs`: Perubahan dokumentasi
- `style`: Perubahan format, tidak mempengaruhi logika
- `refactor`: Perubahan yang tidak menambah fitur baru maupun memperbaiki bug
- `test`: Menambahkan atau memperbaiki test
- `chore`: Perubahan yang tidak terkait dengan produksi

## Contoh

```
feat(auth): tambah login dengan Google
fix(api): perbaiki endpoint film terbaru
docs(readme): perbarui instruksi instalasi
```
